## Description
Python script to scrape SaaS product reviews from G2, Capterra, and TrustRadius
based on company name and time period.

## Requirements
- Python 3
- requests
- beautifulsoup4

## Installation
pip install requests beautifulsoup4

## Usage
python scraper.py

## Inputs
- Company slug
- Source
- Start date
- End date

## Output
JSON file containing reviews.

## Bonus
Integrated TrustRadius as a third review source.
